1. Baixa o python decentemente pra evitar BO, o pip vem junto: https://www.python.org/ftp/python/3.11.4/python-3.11.4-amd64.exe
2. Depois para as bibliotecas:
python -m pip install flask
python -m pip install flask-dropzone
python -m pip install bcrypt

3. Pra rodar: abre o terminal na pasta que ta o backend.py e digita python backend.py no terminal
